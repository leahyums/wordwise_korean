Privacy Policy

Effective date: February 21, 2026

WordWise Korean ("the extension") is a free, open-source browser extension. This policy explains what data the extension does and does not handle.

Data Collection

The extension does not collect, transmit, or share any personal data. Specifically:

  • No browsing history is recorded or stored.
  • No page content is sent to any server.
  • No analytics or telemetry of any kind is gathered.
  • No third-party services or external APIs are contacted.

Local Processing Only

All annotation logic runs entirely on your device. The vocabulary database (6,065 TOPIK words) is bundled inside the extension package and never fetched from the internet.

User Settings

Your preferences (vocabulary level, font size, highlight toggle) are saved using Chrome's built-in chrome.storage.sync API. This syncs settings across your own signed-in Chrome devices only. The developer has no access to this data.

Permissions

  • storage — saves your preferences locally/synced.
  • activeTab — injects translations into the page you are currently viewing.
  • Host permissions (<all_urls>) — enables annotations on any Korean-language website you choose to visit.

Remote Code

The extension does not load or execute any remote code. All JavaScript is bundled at build time and included in the extension package reviewed by the Chrome Web Store.

Children's Privacy

The extension does not collect any data from any user, including children under 13.

Changes to This Policy

If this policy changes, the updated version will be posted at this URL with a revised effective date.
